# Cookie Clicker - Jules

A simple, beautifully designed Cookie Clicker web application built with vanilla HTML, CSS, and JavaScript.

## Features
- **Modern Aesthetics**: Features a sleek dark-mode inspired glassmorphism UI with a smooth gradient background.
- **Interactive Gameplay**: A clickable cookie with responsive transform animations.
- **Particle Effects**: Dynamic floating "+1" animations trigger wherever you click the cookie.
- **Real-time Counter**: A large, easy-to-read counter that updates instantly with pop animations.
- **Persistence**: Your score is saved in a browser cookie, so it won't be lost when you refresh the page.
- **Separation of Concerns**: Logic is organized in a dedicated `script.js` file.

## How to Run
Since it is a purely static website, no build process or server is required. 
1. Open the `Jules_test` folder.
2. Double click the `index.html` file to open it in your browser.
3. Start clicking! 🍪
