# Cookie Clicker

A simple web page with a cookie that you can click to increase a counter.

## Features
- A big cookie in the middle of the screen.
- A click counter that increments every time you click the cookie.
- **Persistence**: Your click count is saved in a browser cookie and will persist even if you reload the page.
- Simple and clean UI.

## File Structure
- `index.html`: The main page structure and styling.
- `script.js`: The application logic and persistence management.

## How to use
Just open `index.html` in your web browser and start clicking!
